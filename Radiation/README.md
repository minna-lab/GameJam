++++++++Radiation 
Jeu réalisé en JavaScript / HTML / CSS dans le cadre du Dangerous JS Jam.


++++++++Concept 
Vous incarnez une femme qui se réveille dans la réception d’un hôtel abandonné après une catastrophe nucléaire.

L’immeuble est contaminé :
tout ce que vous touchez peut être dangereux.

Le joueur doit explorer les étages de l’immeuble, trouver des ressources et survivre le plus longtemps possible.


+++++++++Le Jeu 
La difficulté augmente à chaque étape
-> objectifs 
+survivre 
+monter le plus haut possible
+éviter les objests radioactifs
+comprendre ce qui s'est passé dans cet hôtel


+++++++++Gameplay 

/Le joueur peut:
-explorer les pièces
-interagir avec des objets
-trouver des indices
-récupérer des ressources

/Chaque interaction peut produire différents effets:
-récupérer de la vie 
-augmenter la radiation 
-déclencher un piège 
-découvrir un indice 

/Plus le joueur monte dans l'immeuble:
-la radiation augmente
-les objets deviennent plus dangereux
-l'environnement devient plus instable 


+++++++++Technologies utilisées 

-HTML structure du jeu 
-CSS ambiance visuelle (lumière,glitch,radioactivité)
-JavaScript logique du jeu 

++++++++Controles

Point & Click 
-> cliquer pour se déplacer 
-> cliquer sur les objets pour interargir 
-> flèches directionnelles pour bouger 


+++++++++Structure du jeu 

Etage 0: Réception /pieces 
Etage 1: couloirs
etage 2 maintenance 
etage 3 laboratoire 
etape supérieur zones générées avec dangers aléatoires 

Chaque étage contient: 
-objets intéractifs 
-pièges
-ressources 
-indices narratifs
